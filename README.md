# Banking-Customer-Analytics-Dashboard

## Project Overview
This project analyzes banking customer data to identify trends in customer accounts, loan amounts, and account growth.
The final project is an interactive Tableau dashboard designed to summarize key business metrics.

## Tools Used
- Tableau Public
- SQL
- Excel

## Skills Demonstrated
- Data Exploration
- Data Cleaning & Validation
- SQL Querying
- Exploratory Data Analysis
- KPI Development
- Dashboard Design
- Data Visualization
- Business Analytics
- Tableau Public
- Microsoft Excel

## Dashboard Features
- Total Customers KPI
- Average Account Balance KPI
- Average Loan Amount KPI
- Customers by Account Type
- Average Account Balance by Account Type
- Average Loan Amount by Loan Type
- Top 10 Cities
- Accounts Opened by Year
